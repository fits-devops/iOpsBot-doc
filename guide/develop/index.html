接口说明 ​

如何对接 iOpsBot？🔗 ​

iOpsBot 基于 Swagger 生成文档，你可以通过调用 API 来实现与 iOpsBot 的交互。

  1. 获取 Token 信息
  2. 调用 API 接口

1. 获取 Token 信息 ​

通过菜单「用户设置」- 「API 密钥」 创建 并 获取 Token 信息。

2. 调用 API 接口 ​

接口文档

http://192.168.0.123:5050/docs

接口调用示例 ​

创建新的 Chat ​

POST /chat

  • 请求 Header：-H "Authorization: Bearer 你的 Token"

  • 请求 Body：

json
{
  "name": "会话名称"
}
  • 响应 Body：
json
{
  "chat_id": "d603bb57-af34-496f-8985-8e43bf2a2c89",
  "user_id": "c546a3d7-12c6-45ed-b840-05d49b392554",
  "creation_time": "2024-03-13T06:49:33.838025",
  "history": null,
  "chat_name": "会话名称"
}

开始对话调用 ​

POST /chat/{chat_id}/question/stream

  • 请求 Header：-H "Authorization: Bearer 你的 Token"

  • 请求 参数

    参数名参数类型是否必填说明
    chat_idstring是会话 ID
    brain_idstring否场景 ID

  • 请求 Body：

json
{
  "question": "hello",
  "model": "Qwen-7B-Chat",
  "temperature": 0,
  "max_tokens": 2000,
  "brain_id": "f16cfbe1-7ecf-4f2c-a1e3-d6b364d894b7"
}
  • 响应 Body：
data: {
    "chat_id": "da8a70f8-fd86-4954-ba26-a8717eade8dd",
    "message_id": "51ab7c15-547c-4d9b-81a7-2c6d4478bd80",
