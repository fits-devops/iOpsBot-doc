快速上手 ​

欢迎来到 iOpsBot 👋 ​

iOpsBot 是一款基于 Quivr 开源应用 + Qwen 模型开发的智能运维助手，它能够自动识别和处理运维工作中的常见问题，并提供相应的解决方案。使用 iOpsBot，运维人员可以节省大量的时间和精力，提高工作效率。

如何使用 iOpsBot？ 🚀 ​

您可以使用 iOpsBot 提供的 API 或者直接访问 iOpsBot 的 Web 界面来使用它。

  • 🔍 在文件中搜索：能够搜索您提供的文件，并返回与搜索内容相关的结果。
  • 🔀 交互式搜索：提供一个交互式的搜索界面，让您能够更方便地搜索相关信息。
  • 🧠 储存您的知识：将您所有的知识储存在一个地方，以便在需要时快速检索。
  • 🔃 分享您的知识：将您的知识分享给他人，共同进步。

特点 🎆 ​

公共和私有场景 ​

您可以创建一个公共的 iOpsBot 场景，供所有人使用，也可以创建一个私有的场景，只供特定用户使用。

  • 公共的：公共场景是所有人都可以访问的，适合用于共享知识或者进行知识交流。
  • 私有的：私有场景是只有特定用户才能访问的，适合用于个人知识管理或者团队协作。

您可以使用电子邮箱账号和密码登录来创建和管理您的场景。

向助手提出问题 ​

您可以向助手提出您的问题，助手将根据您的问题类型和内容，返回相应的答案。

  • 第一步：访问 Web 界面，前往iOpsBot并登录您的账户。
  • 第二步：通过@选择一个场景，也可以直接输入问题,直接与模型对话。
  • 第三步：输入您的问题，Enter 回车提交。
  • 第四步：等待答案返回，查看答案并使用。

    请记住，iOpsBot 的智能程度取决于您所提供的知识。如果它不知道问题的答案，请考虑向您的知识库添加更多信息。

个性化定制 ​

您可以根据自己的需求，定制 iOpsBot 的提示词。

  • 第一步：访问 Web 界面，前往「场景」-「管理场景」菜单，选择您要编辑的场景。
  • 第二步：在「设置」选项卡中，编辑自定义提示"选择小助手的性格"。点击"保存"按钮以保存更改。
